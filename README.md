# PlayWith - Minecraft サーバー紹介サイト

本リポジトリは、かつて友人と共同運営していた **Minecraft サーバー「PlayWith」** の紹介用に制作した  
**静的 Web サイト（HTML / CSS）** のコードを保管するためのものです。

当サイトは趣味の範囲内で、受験期に約 1 週間ほどでデザインから実装まで行いました。  
ロゴとチーム名を除き、デザイン・コーディングはすべて自作です。

---

## 🔍 サイトの目的

- 友人向けに Minecraft サーバーの案内ページを提供すること
- サーバー参加用の情報を一箇所に集約し、わかりやすく提示すること
- サーバーの雰囲気や特徴をシンプルに伝えること

---

## 📚 掲載していたコンテンツ

当時サイトに掲載していた主な情報は以下の通りです：

- **サーバー招待コード**
- **サーバー情報（バージョン、接続方法など）**
- **Discord 招待コード**
- **運営からのお知らせ（簡易的）**

※現在サーバーは解散しているため、実際の招待コード・接続情報は削除済みです。

---

## 🛠 技術スタック（Tech Stack）

- **HTML5**
- **CSS3**
- 完全静的サイト（JavaScript 依存なし）
- 自作 Web デザイン（レイアウト・色設計・UI すべて自作）

公開にはレンタルサーバー（スターサーバー）を利用していました。

---

## 📁 構成について

本リポジトリには、当時の静的サイトの以下のファイルが含まれます：

- `index.html`（トップページ）
- `style.css`（デザイン）
- `assets/`（ロゴなどの静的ファイル）
- その他必要な静的コンテンツ
